## 数据源说明

### 1. 聚合数据（推荐）
**当前使用**：65,946个监测点
**速度范围**：-579.6 ~ 504.8 mm/yr（已修正单位）
**数据来源**：你的Month2项目

```
Month2项目数据
└── mintpy/geo/geo_velocity.h5 (原始: -0.725 ~ 1.254 m/year)
        ↓ gis_tools.py 转换
Month3聚合数据
└── data/processed/beijing_velocity_aggregated.shp
    ├── 65,946个聚合点
    ├── 网格大小: 0.005度 (~500m)
    └── 每个点包含: 速度均值/标准差/最小值/最大值/点数
```

### 2. 采样数据
**当前状态**：97,224个监测点
**速度范围**：-0.6 ~ 0.5 mm/yr（旧数据，单位未转换）

## 客户如何导入自己的数据？

### 方法1：替换Shapefile（推荐，最简单）

**步骤**：

1. 准备你的Shapefile，必须包含以下字段：
```
必需字段:
- longitude: 经度 (WGS84)
- latitude: 纬度 (WGS84)
- velocity: 沉降速率 (mm/year)

可选字段:
- velocity_std: 速度标准差
- velocity_min: 最小速度
- velocity_max: 最大速度
- point_count: 聚合点数
```

2. 将你的Shapefile放到：
```
data/processed/
└── your_data.shp (替换 beijing_velocity_aggregated.shp)
```

3. 修改配置文件 `config/config.py`：
```python
VELOCITY_FILE = PROCESSED_DATA_DIR / "your_data.shp"
```

4. 重启应用

**Shapefile格式示例**：
| longitude | latitude | velocity | geometry |
|-----------|----------|----------|----------|
| 116.4074 | 39.9042  | -25.3    | POINT(...) |
| 116.4081 | 39.9050  | -12.8    | POINT(...) |

---

### 方法2：从HDF5转换（适合InSAR用户）

如果你的数据是MintPy输出的HDF5格式：

1. 将HDF5文件放到项目目录
2. 运行转换脚本：
```bash
python src/utils/gis_tools.py \
    --mode aggregate \
    --grid-size 0.005 \
    --output your_data.shp
```

3. 重启应用

---

### 方法3：修改代码读取CSV（最灵活）

如果客户只有CSV文件：

**CSV格式示例**：
```csv
lon,lat,velocity
116.4074,39.9042,-25.3
116.4081,39.9050,-12.8
...
```

在 `app.py` 中添加：
```python
# CSV加载函数
@st.cache_data
def load_csv_data(csv_path):
    df = pd.read_csv(csv_path)
    gdf = gpd.GeoDataFrame(
        df,
        geometry=gpd.points_from_xy(df.lon, df.lat),
        crs="EPSG:4326"
    )
    return gdf
```

## 数据格式要求

### 坐标系
- **必须使用 WGS84 (EPSG:4326)**
- 经度范围：-180 ~ 180
- 纬度范围：-90 ~ 90

### 沉降速率单位
- **必须使用 mm/year**
- 负值 = 沉降
- 正值 = 抬升

### 数据量建议
| 数据量 | 加载时间 | 推荐场景 |
|--------|----------|----------|
| < 10万点 | < 5秒 | 小区域监测 |
| 10-50万点 | 5-15秒 | 中等城市 |
| > 50万点 | > 15秒 | 大型城市（建议聚合）|

## 快速测试数据

如果想快速测试，可以用这个Python脚本生成模拟数据：

```python
import numpy as np
import pandas as pd
import geopandas as gpd
from shapely.geometry import Point

# 生成模拟数据
np.random.seed(42)
n_points = 10000

lon = np.random.uniform(116.0, 116.5, n_points)
lat = np.random.uniform(39.8, 40.0, n_points)
velocity = np.random.normal(-50, 100, n_points)  # mm/yr

gdf = gpd.GeoDataFrame(
    {'longitude': lon, 'latitude': lat, 'velocity': velocity},
    geometry=gpd.points_from_xy(lon, lat),
    crs="EPSG:4326"
)

gdf.to_file('data/processed/test_data.shp')
```
