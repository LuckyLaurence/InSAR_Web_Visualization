### 3. 风险分级算法
**文件**: `spatial_analysis.py`

```python
def get_risk_level(row):
    vel = row['velocity_mean']
    if vel < -200:  # 严重沉降
        return 'high'
    elif vel < -50:  # 明显沉降
        return 'medium'
    else:
        return 'low'
```
