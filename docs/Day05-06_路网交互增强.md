### 2. 路网图层集成
**文件**: `src/app.py`

#### 关键修复
1. **坐标匹配问题**
   - 原问题：路网坐标（116.0~116.5）与InSAR数据（114.1~115.5）不匹配
   - 解决方案：更新create_mock_roads.py，使用InSAR实际坐标范围

2. **图层顺序**
   - 路网图层先添加，InSAR点图层后添加
   - 确保路网在点上方可见

3. **样式优化**

### 3. 交互控制
- 侧边栏开关：显示/隐藏路网
- 侧边栏开关：显示/隐藏沉降热点
- 鼠标悬停高亮道路

### 4. 沉降热点分析
- 明显沉降点（速率 < -0.5 mm/yr）：红色
- 轻微沉降点（-0.5 ~ -0.2 mm/yr）：橙色
- 稳定点：绿色

## 技术要点

### PyDeck图层叠加顺序
```python
layers = []
# 先添加路网（底层）
if show_road_network:
    layers.append(road_layer)
# 后添加InSAR点（顶层）
layers.append(insar_layer)
```

### 坐标数据转换
```python
# GeoDataFrame -> PyDeck格式
road_data = []
for idx, row in gdf_roads.iterrows():
    coords = list(row.geometry.coords)
    coords_numeric = [[float(c[0]), float(c[1])] for c in coords]
    road_data.append({
        'path': coords_numeric,
        'name': str(row.get('name', f'道路{idx}')),
        'type': str(row.get('highway', 'unknown'))
    })
```

## 效果展示
- 路网以蓝色线条叠加在InSAR速度场上
- 道路宽度8像素，悬停时高亮为红色
- 沉降热点以不同颜色标识风险等级
