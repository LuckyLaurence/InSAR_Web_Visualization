## 技术实现

### 数据流程
```
Shapefile → GeoDataFrame → PyDeck Layer → 3D Map
```

### 颜色编码方案

| 速度范围 (mm/yr) | 颜色 | 类别 |
|------------------|------|------|
| < -100 | 红色 | 严重沉降 |
| -100 ~ -50 | 橙色 | 明显沉降 |
| -50 ~ -20 | 黄色 | 中度沉降 |
| -20 ~ +20 | 绿色 | 稳定 |
| > +20 | 蓝色 | 抬升 |

### 地图配置
```python
view_state = pdk.ViewState(
    latitude=39.9,      # 北京中心
    longitude=116.4,
    zoom=10,            # 缩放级别
    pitch=45,           # 俯视角
    bearing=0           # 方位角
)
```

## 遇到的问题与解决

### 问题1: Shapefile字段名截断
- **现象**: `velocity_mean` 被截断为 `velocity_m`
- **解决**: 检测并重命名字段

### 问题2: PyDeck坐标格式
- **现象**: 地图不显示数据
- **解决**: 确保列名为 `lon` 和 `lat`

### 问题3: 颜色编码
- **现象**: 颜色无法反映速度范围
- **解决**: 创建 `get_color()` 函数动态计算RGB值
